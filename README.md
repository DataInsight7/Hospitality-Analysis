# Hospitality-Analysis
Overview

A data analytics project focused on evaluating revenue, occupancy, and booking performance in the hospitality domain using Excel, SQL, Power BI, and Tableau.

Objective

Identify revenue drivers and occupancy gaps

Analyze cancellations and booking behavior

Support data-driven operational decisions

Tools

Excel | SQL | Power BI | Tableau

KPIs

Total Revenue

Occupancy Rate

Cancellation Rate

Booking Volume

Revenue by City, Property & Room Type

Weekday vs Weekend Revenue

Key Insights

Mumbai is the top revenue-generating city

RT2 & RT3 room categories contribute the highest revenue

~25% booking cancellation rate impacts revenue stability

Occupancy levels are below optimal

Weekday demand is significantly higher than weekends
